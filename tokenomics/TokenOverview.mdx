# $LAM Tokenomics

The Large Action Model Token ($LAM) is the native utility token that powers the Action Model ecosystem. It is designed to facilitate a circular economy, rewarding community contributors while creating a sustainable model for enterprise use.

This section provides a detailed breakdown of the token's role, from earning rewards to powering the API and contributing to the deflationary model.
