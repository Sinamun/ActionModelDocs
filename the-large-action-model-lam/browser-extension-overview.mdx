## What Data Is Collected

### Comprehensive Yet Respectful

<Info>

**Transparency First**: We collect only what's necessary to train the LAM effectively. Every piece of data serves a purpose in teaching AI how to navigate the digital world.

</Info>

<AccordionGroup>
<Accordion title="DOM Elements" icon="code">

**What**: HTML structure, element IDs, classes, and attributes
**Why**: Identifies interactive elements and their relationships
**Example**: Button locations, form fields, navigation menus

</Accordion>

<Accordion title="Screenshots" icon="camera">

**What**: Visual captures of page states (anonymized)
**Why**: Provides visual context for actions
**Example**: Layout understanding, visual element recognition

</Accordion>

<Accordion title="User Actions" icon="mouse-pointer">

**What**: Clicks, scrolls, hovers, and keyboard inputs
**Why**: Maps action sequences to outcomes
**Example**: How users complete specific tasks

</Accordion>

<Accordion title="URL Navigation" icon="route">

**What**: Page transitions and navigation paths
**Why**: Understands site structure and user journeys
**Example**: Multi-step checkout processes

</Accordion>

<Accordion title="Timing Data" icon="stopwatch">

**What**: Time between actions and page load times
**Why**: Creates realistic action pacing
**Example**: Natural typing speed, thinking pauses

</Accordion>

<Accordion title="Network Requests" icon="network-wired">

**What**: API calls and responses (metadata only)
**Why**: Understands dynamic content loading
**Example**: AJAX updates, form submissions

</Accordion>
</AccordionGroup>

## FAQs

<AccordionGroup>
<Accordion title="How much can I realistically earn?">

**Passive users** average 200-500 $LAM per month. **Active trainers** can earn 5,000-20,000 $LAM monthly, especially when focusing on high-bounty platforms.

</Accordion>

<Accordion title="Does it slow down my browsing?">

No. The extension uses minimal resources (&lt;1% CPU, &lt;50MB RAM) and operates asynchronously to never impact your browsing experience.

</Accordion>

<Accordion title="Can I use it at work?">

Yes, but check your company's policies. Many businesses encourage it as they benefit from the improved automation capabilities.

</Accordion>

<Accordion title="What about my passwords and personal data?">

Never captured. The extension automatically excludes password fields, payment information, and personal data entry.

</Accordion>

<Accordion title="Can I pause it temporarily?">

Yes. Click the extension icon to pause/resume anytime. You can also set schedules for automatic pausing.

</Accordion>
</AccordionGroup>
