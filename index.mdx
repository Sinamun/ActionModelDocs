# Welcome to Action Model

Action Model is the world’s first **Large Action Model (LAM)**—a new AI paradigm that moves beyond language to take human-like actions across any software interface. Our community-owned model learns from user demonstrations to automate complex workflows on any website or application.

This documentation is your complete guide to using our tools, contributing data, and building on the future of AI-powered automation.
